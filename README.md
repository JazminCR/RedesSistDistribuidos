# Redes y Sistemas Distribuidos

Laboratorios de la materia **Redes y Sistemas Distribuidos** (3º año – Lic. en Ciencias de la Computación, FaMAF).

## Contenido
- **Lab 0 – Cliente HTTP**
  - Cliente HTTP (`hget`) en Python 3
  - Descarga de páginas web mediante el protocolo HTTP (RFC 2616)

- **Lab 1 – API REST**
  - Desarrollo de una API en **Python + Flask**
  - Gestión de base de datos de películas
  - Integración con una API externa
  - Evaluación de respuestas, códigos HTTP, escalabilidad y seguridad

- **Lab 2 – Servidor HFTP**
  - Implementación de un protocolo de transferencia de archivos casero
  - Protocolo HFTP sobre TCP

- **Lab 3 – Capa de Transporte**
  - Modelo de colas usando **OMNeT++**
  - Análisis de tráfico, buffers y tasas de servicio

- **Lab 4 – Capa de Red**
  - Modelo de red en anillo
  - Implementación de enrutamiento entre nodos
  - Simulación de tráfico y decisión de interfaces

Repositorio con fines académicos.
